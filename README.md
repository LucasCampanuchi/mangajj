# Manga JJ - App

App to read mangas online and offline using [MangaJJ API](https://github.com/jjeanjacques10/mangajj)

## Tecnologias 

- Flutter
- MobX
- API

# Get Started

```bash
# Clone
$ git clone https://github.com/LucasCampanuchi/mangajj.git

$ flutter pub get
$ flutter run
# ou utilize o VSCode para executar o projeto
```
